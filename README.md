# SATAS-MERN 🛒

This is a Web App, developed under the  **MVC** design pattern, using the **MERN** stack. 

## Features 🔥
- User role management 🦸‍♂️
- Stock management 🛍
- Dashboard 📈

## Technologies 🛠
- MongoDB 🍃
- Express 🤍
- Nodejs 💚
- Reactjs 💙
